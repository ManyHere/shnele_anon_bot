# shnele_anon_bot
Данный проект представляет собой анонимный чат-бот для мессенджера Telegram. 
Бот предназначен для соединения случайных пользователей в анонимные диалоги без передачи персональной информации.

Работа бота основана на очереди ожидания: пользователь может начать поиск собеседника, дождаться подключения или отменить поиск. 
При наличии свободного пользователя бот автоматически соединяет их в диалог.

В процессе общения бот пересылает все типы сообщений, включая текстовые и мультимедийные (фото, видео, голосовые сообщения, стикеры и документы). 
Команды управления диалогом не передаются собеседнику и обрабатываются только ботом.

Бот предоставляет команды для управления общением:
- завершение текущего диалога;
- переход к следующему собеседнику;
- просмотр общей статистики сообщений пользователя.

Для хранения статистики используется локальная база данных SQLite, в которой сохраняется идентификатор пользователя и общее количество отправленных им сообщений.

Проект реализован на языке Python с использованием асинхронной библиотеки aiogram и может быть запущен локально без использования сторонних серверов.
